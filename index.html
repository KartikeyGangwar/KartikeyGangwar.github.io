Kartikey Singh

Undergraduate Researcher — Scientific Machine Learning, Physics-Informed Neural Networks, Nonlinear PDEs

I work on scientific machine learning and numerical methods for nonlinear partial differential equations. My research introduced Solution Manifold Incompatibility (SMI), a structural limitation of linear multi-fidelity physics-informed neural networks.

My approach is implementation-first: I build computational systems, stress-test assumptions empirically, and then formalize the observed behavior mathematically.

Research

Solution Manifold Incompatibility in Multi-Fidelity PINNs

  • Identified systematic failure of linear fusion for nonlinear PDEs
  • Formalized operator non-closure and tangency failure
  • Validated using PyTorch PINNs and a custom CFD solver (Re = 1000)
  • Published with permanent DOI and open-source code
